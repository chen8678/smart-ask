# Apple Bezel 素材包说明

为方便比赛提交及对外展示，以下整理了一套标准化的 Bezel 截图流程与素材分区，便于在金融、医疗及学习分析页面中快速输出苹果风格的展示图。

## 1. 画布基线
- **分辨率**：2880 × 1800（16:10），适配 MacBook Pro 14/16 英寸。
- **安全区域**：上下各留 120 px，左右各留 160 px，避免导航与系统状态遮挡。
- **背景**：#050505 ~ #0A0A0A 的径向渐变，透明度 0.85，叠加 15% 模糊。

## 2. 模块分层
| 图层 | 说明 |
| --- | --- |
| Layer 1 | Liquid 背景（径向/线性渐变，参考 `Login.vue` 方案） |
| Layer 2 | 玻璃片 (`var(--apple-glass)`) 与阴影（Y 轴 40 px，模糊 80 px，透明度 0.35） |
| Layer 3 | 真实页面截图（1920 × 1200，缩放 73%） |
| Layer 4 | 前景高光/Logo/SF Symbols |

## 3. 页面推荐组合
1. **学习分析主看板**：`LearningAnalytics` 顶部导航 + CourseAnalytics 卡片。
2. **行业模块**：金融风险 + 医疗诊断，并列 2-up 布局，距中心 ±480 px。
3. **认证/配置**：`AIModelConfig` + `Login` 玻璃风格，用于展示系统配置能力。

## 4. 导出流程
1. 在 `src/views` 中打开目标页面，使用浏览器 1440px 宽度截图（隐藏浏览器 UI）。
2. 导入 Figma/Sketch，按上表分层贴入。
3. 套用「MacBook Pro Bezel」组件：阴影模式 Soft Light，透明度 0.65。
4. 导出 PNG（@1x 用于 PPT，@2x 用于 Keynote / 官网）。

> 提示：如需快速生成不同配色，可直接复用 `Login.vue` 中的渐变定义与 `.ghost-btn` 组件，保持黑白底 + 彩色高光的一致性。
